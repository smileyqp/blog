人生中第一次说走就走的旅行

发表于 2017-08-18 | 更新于 2018-06-08 | 分类于 日记

今天去下沙的建设银行办理保险领钱手续，好远啊，坐公交和地铁，天气还那么热。。

终于办好了手续，此时已经下午两点多了，好不想回学校啊。。好想出去走走啊。。。

于是就有了人生中第一次说走就走的旅行~

一直想去南京，但是此时杭州去南京的票没了，打开去哪儿旅行，看看附近的景点，看上了西塘，好像很近的样子，就在嘉兴！杭州过去就几站吧。。于是直接在地铁上转去了火车站，去哪儿网上买好去的车票，顺便在西塘附近订了一家青年旅舍，当时的心情是有点紧张、激动，但是感觉住宿定好了就不用慌了，然后此时手机快没电了，我带了钱包出来居然没带充电宝!希望能撑到青年旅舍吧。。虽然感觉不太可行。
到了东站，发现了共享充电宝，只需要扫码，然后信用分足够就可以借走。。这不是在帮我吗？有种雪中送炭的感觉呢。。。应该刚出来不久的东西吧，反正我是第一次见，杭州真是一个发展迅速的城市啊。。
到了嘉兴，有点晚了，而且我坐公交车在转车那一站做过一站了，错过了最后一班车，。。我在公交站坐了十多分钟，路上没什么人，虽然才八点，不过那个地方应该比较偏僻吧，打顺风车没打到，最后只能叫快车了，超过了我一晚上的住宿费呢。。我那天晚上的住宿费才57块钱。。上车后，司机越开越偏，他了解到我是一个人来的，问我这么晚了居然不怕，本来不怕的，他这么一说，就怕起来了，然后连忙编着说明天有朋友一起玩的。到了”落脚点”,这是客栈名字哈哈，已经十点了，老板看我一个人有点小惊讶呢，给我介绍了西塘所有可以玩的，还有酒吧清吧啥的，桥啊等一系列的景点~然后，他给我安排了大床房，一个人住！哇塞~感觉一天的不愉快都消失了呢！
进门看了看房间，粉红色的蚊帐哎，还有沙发和电视机，好浪漫的说~感觉要今晚出去玩玩才能对得住我从花了一个多小时公交+快车来到这里的辛酸，于是带上店主给我的指南图纸就出门了~
这个时间只有酒吧和一些饰品店还开着门，小吃啥的已经没有了，不过烧烤和热狗啥的还是有的，于是一边吃一边逛，酒吧大概分两种，一种是很热闹很嗨，会发出震耳欲聋的声音，一般遇到这种我就快跑过去了，然后另一种叫清吧，是很安静的，喝酒喝茶的地方，然后也有一些歌手在酒吧里唱歌，不过我不是特别感兴趣，也就稍微走了一圈，算是为明天踩踩点吧，出景区的时候已经快十二点了，也没什么人了，还有两个女孩子找我问路，路过一家烧烤摊买了一些吃的才回去客栈，想想有点后怕。
第二天，早早起床了，开始了一天的游玩，其实我很喜欢这里古风古味的感觉，小桥流水，很安静，虽然景区里人有点多，但是一点也不吵闹，我就各种店都进去逛了逛，然后发生了一件疑似被骗钱的事情。
走到一个店里买了一个姓氏的挂件准备送给我闺蜜来着，服务员叫我抽奖，然后抽到了一等奖，也就是全场珠宝可以一折买下，她显得非常震惊，而我也非常开心，当时选了一下，还真看上了一条，是和田玉，然后我想到我闺蜜，还打算再买一条，然后询问了店员，她说最多买两条。。。我就又选了一条。。一条四千多吧。。。一折的话是四百多，当时觉得好值。。。现在想想有点傻了。。然后我就一下子挑了两条，还好算是那里性价比最高的两条项链了，而且保证是正品，买下来的时候还是特别开心的，后来和朋友说的时候，她说我被骗了，说这种店都这么骗人的，果然上网一查这种新闻
遍地都有。。。有点伤心，但是我在淘宝上看了一下，这种项链其实也基本要这个价格，而且还没有我的款，想想还行吧，亏的话应该也只是亏一百左右，没事没事。。。
然后这场旅行就结束了，买了一份芡实糕，算是那里的特产吧，回到学校还是很开心的，毕竟出去玩了一圈古镇，住了一次客栈，还买到了喜欢的项链，而且闺蜜也很喜欢准备送她的项链呢。
